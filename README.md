### css 폴더 구조 설명
파일명 | 파일 설명
--- | --- |
font.css | 웹 폰트가 담긴 곳입니다. |
reset.css | User Agent Stylesheet를 리셋해주는 파일입니다. <br> 에릭 마이어의 reset css를 사용하여 모든 요소의 마진, 패딩을 0으로 만들고, 기본 서식을 제거해 주었습니다.|
style.css | PC 스타일과 모바일용 스타일을 포함한 모든 스타일이 포함된 곳입니다. |
